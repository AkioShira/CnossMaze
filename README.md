# CnossMaze
<h4>Описание:</h4> Программа, генерирующая лабиринт и путь его прохождения. Курсовой проект по "технологиям программирования" Шишко А.А. СКС-15 </br>
Для генерации лабиринта был использован алгоритм Эйлера. Для нахождения выхода из него был использован волновой алгоритм. </br>
<h4>Дата:</h4> 26.11.2017
